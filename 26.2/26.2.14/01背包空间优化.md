## 01 - 背包空间优化
### 原代码
```cpp
    // 问题1：
    for (int i = 1; i <= n; i++)
    {
        for (int j = 0; j <= V; j++)
        {
            // 不选i
            dp[i][j] = dp[i - 1][j];

            // 选（如果能）
            if (j - v[i] >= 0)
            {
                dp[i][j] = max(dp[i][j], dp[i - 1][j - v[i]] + w[i]);
            }
        }
    }
```
### 优化后
```cpp
   for (int i = 1; i <= n; i++)
    {
        for (int j = m; j >= v[i]; j--)
        {
            // // 不选i
            // dp[j] = dp[j];
            // 选
                dp[j] = max(dp[j], dp[j - v[i]] + w[i]);
        }
    }

```

修改了 `遍历顺序` 并且在遍历过程中直接剔除了 `j < v[i]` 不够装的情况
