# 完全背包空间优化

## 原代码
```cpp
for (int i = 1; i <= n; i++)
    {
        for (int j = 0; j <= m; j++)
        {
            f[i][j] = f[i - 1][j];
            if (j - v[i] >= 0)
            {
                f[i][j] = max(f[i][j], f[i][j - v[i]] + w[i]);
            }
        }
    }
```

## 优化后
```cpp
memset(f, -0x3f3f, sizeof f);
f[0] = 0;
for (int i = 1; i <= n; i++)
    {
        for (int j = v[i]; j <= m; j++)
        {
            f[j] = max(f[j], f[j - v[i]] + w[i]);
        }
    }
```

## 总结

 > 01背包：第二层for循环 -> 从大到小

 > 完全背包：第二层for循环 -> 从小到大

主要区别在二者状态转移方程  
完全背包 `f[i][j] = max(f[i][j], f[i][j - v[i]] + w[i])`用的是 `i` 而不是 `i-1`
