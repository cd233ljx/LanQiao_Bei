## 洛谷 P1908-逆序对
### 逆序对定义 同线性代数
#### 利用（分治）归并排序优化时间复杂度
- **共分成三部分**
1. 统计全都在左边的
```cpp
    ret += merge(left, mid);
```
2. 统计全都在右边的
```cpp
    ret += merge(mid + 1, right);
```
3. 统计一左一右
- 引理：左右都排序后再计算不影响逆序对的结果
```cpp
int cur1 = left, cur2 = mid + 1, i = left;

    while (cur1 <= mid && cur2 <= right)
    {
        if (a[cur1] <= a[cur2])
        {
            tmp[i++] = a[cur1++];
        }
        else
        {
            // 在合并两个有序数组过程加上统计逆序对结果
            ret += mid - cur1 + 1;

            tmp[i++] = a[cur2++];   
        }
    }

    while (cur1 <= mid)
        tmp[i++] = a[cur1++];
    while (cur2 <= right)
        tmp[i++] = a[cur2++];

    for (int j = left; j <= right; j++)
        a[j] = tmp[j];

```
