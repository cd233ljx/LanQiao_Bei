
# 设置相关笔记



## 一、Code-Runner:

### 1. 在 VS Code 中能完美解决 `C/C++` 使用 **Code-Runner** 中文乱码问题：

#### 使用 chcp 65001 指令  
   - `chcp 65001` 指令会设置当前终端编码为 UTF-8，后续命令才能正确输出中文。

####  完整配置步骤（VSCode 中）
1. 打开 VSCode 设置（`Ctrl + ,`）
2. 搜索 `Code Runner` → 点击 `在 settings.json 中编辑`
3. **替换为以下完整配置**：
```json
  "code-runner.executorMap": {
    "c": "chcp 65001 && cd $dir && gcc $fileName -o $fileNameWithoutExt && $dir$fileNameWithoutExt",
    "cpp": "chcp 65001 && cd $dir && g++ $fileName -o $fileNameWithoutExt && $dir$fileNameWithoutExt",
  }
```


### 2. 解决 Code-Runner 无法在只读编辑器中编辑

#### 在 settings.json 中编辑 或 在 `UI` 处勾选 code-runner.runInTerminal (改为用终端输出)
```json
  "code-runner.runInTerminal": true,
```
